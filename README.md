# Automator

Create automated workflows and scripts for zOS.

## Features

- Visual workflow builder
- Drag-and-drop actions
- File operations (copy, move, rename)
- Text processing
- Shell script execution
- HTTP requests
- Notifications

## Installation

```bash
npm install @zos-apps/automator
```

## Usage

```tsx
import Automator from '@zos-apps/automator';

<Automator />
```
